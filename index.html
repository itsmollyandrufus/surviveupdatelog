Cara Menambahkan NPC Custom Kedalam Mod

Dimana saya bisa download skin npc custom?

Kamu bisa mendownload di web gta5-mods.com.

Skin npc custom seperti apa yang harus saya download?

Usahakan cari skin dengan tipe add-on bukan replace, dan usahakan mencari skin dengan upload date sebelum tahun 2025, karena banyak skin baru yang diupload pada tahun 2025 memiliki bug untuk versi GTA 1.68 build 3095 atau 3179.

Saya sudah download skin npc custom, apa langkah selanjutnya?

Jika sudah mendownload skin npc custom, kamu bisa membuka program OpenIV.



Kemudian pergi ke mods -> update -> x64 -> dlcpacks -> addonpeds -> dlc.rpf -> peds.rpf



Klik edit mode supaya bisa memasukkan file skin.



Masukkan skin npc custom yang telah kamu download kedalam folder peds.rpf di OpenIV.

Saya sudah memasukkan file skin custom npc di OpenIV, apa langkah selanjutnya?

Jika file skin custom npc sudah dimasukkan lewat OpenIV, langkah selanjutnya adalah buka program AddonPeds Editor.exe dengan cara klik kanan dan run as administrator. Letak file AddonPeds Editor.exe ada di dalam folder GTA V.

Kemudian klik Peds dan pilih New Ped.



Kemudian pada Model Name ketikkan nama yang sesuai dengan yang tadi dimasukkan kedalam OpenIV (perhatihan huruf kapital dan spasi, semuanya harus cocok).



Untuk Ped Type kamu bisa pilih jenis kelamin skin npc customnya mau male atau female.

Untuk Is Streamed gunakan false jika mod skin yang kamu pasang tadi terdapat 4 file, yaitu dengan format .ydd - .yft - .ymt - .ytd
Jika file modmu tidak terdapat 4 file itu dan terdapat folder didalamnya, maka Is Streamed harus menggunakan false.

Kemudian klik Add Ped.

Langkah terakhir kamu perlu menekan tombol REBUILD.

Saya sudah memasukkan data skin custom di AddonPeds Editor, apa langkah selanjutnya?

Masuk ke folder scripts dan buka file olki_npc_settings.ini menggunakan teks editor seperti notepad atau notepad++.



Kemudian pilih NPC berapa yang ingin kamu ganti modelnya, di tutorial ini saya merubah model untuk NPC3. Cari data Npc3Model1=.



Kemudian ganti nilainya menjadi nama yang sesuai dengan yang tadi dimasukkan di OpenIV dan AddonPeds Editor.



Kamu juga bisa menambahkan model lainnya jika ingin spawn random model, caranya dengan copy paste bagian Npc3Model seperti gambar dibawah ini.



Jangan lupa untuk save notepad dan coba eksekusi webhook untuk spawn npc.

JIKA MODEL YANG BARU DITAMBAHKAN MENGALAMI BUG SEPERTI TEXTURE YANG ANEH, GAME MENJADI CRASH SAAT LOADING SCREEN ATAU GAME CRASH SAAT SPAWN NPC, ITU MENANDAKAN SKIN NPC TIDAK COCOK DAN KAMU HARUS MENGHAPUSNYA DI OPENIV DAN ADDONPEDS EDITOR.
